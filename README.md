# Сайт психолога

## Быстрое размещение в интернете

### Вариант 1: Netlify (самый быстрый - 2 минуты)

1. Откройте https://app.netlify.com/drop
2. Перетащите всю папку с файлами сайта в окно браузера
3. Через несколько секунд вы получите ссылку вида: `https://случайное-имя.netlify.app`
4. Этой ссылкой можно делиться с кем угодно!

### Вариант 2: GitHub Pages

1. Создайте аккаунт на https://github.com (если нет)
2. Создайте новый репозиторий
3. Загрузите все файлы сайта
4. В настройках репозитория включите GitHub Pages
5. Получите ссылку: `https://ваш-username.github.io/название-репозитория`

### Вариант 3: Vercel

1. Зайдите на https://vercel.com
2. Зарегистрируйтесь через GitHub
3. Нажмите "Add New Project"
4. Загрузите папку с файлами
5. Получите ссылку автоматически

## Структура файлов

- `index.html` - Главная страница
- `about.html` - О специалисте
- `services.html` - Услуги
- `pricing.html` - Стоимость
- `contacts.html` - Контакты
- `styles.css` - Стили
- `script.js` - JavaScript

## Локальный просмотр

Просто откройте файл `index.html` в браузере для просмотра на вашем компьютере.
